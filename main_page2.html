Cubes and sound

Programming Assignment

Description

Cubes and sound is a program that draws objects if the user clicks on the screen. If the microphone is enbaled, the objects will be enlarged if there is a lound noise. There are multiple functionalities that are available that can be looked up in the 'examples' section.

Tutorial

There are buttons that will be present on the actual canvas. Click on the button to have the desired effect.
There are options such as changing the shape (2D or 3D), changing the colour, freeze effect, waterfall effect, resume and so on. The buttons are categorized to it is easier to use.
